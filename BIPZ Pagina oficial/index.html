Nuestro Producto

BIPZ es un innovador dispositivo portátil, tu compañero ideal para los viajes en transporte público. Programado para tu tranquilidad, activa una alarma vibratoria y sonora de intensidad moderada para despertarte justo a tiempo, sin molestar a los demás pasajeros. Su uso es intuitivo y directo, diseñado para una experiencia sin complicaciones.

Componentes Utilizados

El cerebro del dispositivo. Procesa los datos del GPS y decide cuándo activar la alarma.

Arduino Uno

Recibe señales de satélite para determinar la ubicación exacta del dispositivo en tiempo real.

Sensor PIR

Genera una vibración silenciosa pero efectiva para despertar al usuario sin causar molestias.

Motor Vibrador

Proporciona la energía necesaria para un funcionamiento prolongado durante varios viajes.

Buzzer

Software Utilizado

Entorno utilizado para programar la lógica del microcontrolador y controlar todos los componentes.

Arduino IDE

Herramienta online para el diseño y la simulación virtual del circuito electrónico antes del montaje físico.

Tinkercad
